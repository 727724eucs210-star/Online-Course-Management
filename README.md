# Online-Course-Management
Online Course Management System built using Spring Boot, REST APIs, JPA, and MySQL, implementing layered architecture with CRUD operations, pagination, and clean service-repository separation.

This application developed using **Spring Boot** that manages online courses, instructors, students, modules, and enrollments.

## Features
- RESTful APIs for CRUD operations
- Layered architecture (Controller, Service, Repository)
- Spring Data JPA integration
- Pagination and sorting support
- Exception handling and validation
- Tested with JUnit and Maven

## Technologies Used
- Java 11 / 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL / H2
- Maven
- REST APIs

## Project Structure
controller → Handles HTTP requests
service → Business logic
repository → Database access
model → Entity classes


## Objective
To build a scalable backend system that efficiently manages online courses and related entities following best practices in Spring Boot development.

## Status
Implemented features up to all Days of the course, including pagination and CRUD operations for instructors, courses, and students.
